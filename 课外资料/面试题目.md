1. 数据类型

   ```java
   public class HelloWord {
       //执行结果是？
       public static void main(String[] args){
          short a =1;
          a = a+1;
          System.out.print(a);
         
          short b =1;
          b +=1;
          System.out.print(b);
       }
   }
   //此时是可以运行的，也不报错。原因如下 
   //java语言规范中关于复合赋值的解释是这样的：E1 op=E2等价于 
   //E1=(T)(E1 op E2),这里的T是E1的数据类型，看到这里 ，大家应该豁然开朗了，原来这个复合赋值是自带了隐式的强制类型转换的。
   ```
